## Development Workflow

### Task Planning
1. Define task in sprint action items with clear deliverables
2. Estimate effort and set target completion date
3. Identify dependencies and prerequisites
4. Assign priority level (High/Medium/Low)

### Implementation
1. Follow development guidelines in `.copilot/patterns/development_guidelines.md`
2. Use ModelManager patterns from `.copilot/patterns/dynamic_model_management.md`
3. Implement comprehensive testing following existing patterns
4. Update documentation and sprint status

### Completion
1. Verify all acceptance criteria met
2. Update sprint action items with completion status
3. Run full test suite to ensure no regressions
4. Document any lessons learned or pattern updates
