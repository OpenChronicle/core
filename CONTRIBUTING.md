# Contributing

OpenChronicle is currently a solo-maintained project. I'm not accepting
pull requests at this time.

## How You Can Help

- **Report bugs.** Open an issue with steps to reproduce, expected
  behavior, and actual behavior.
- **Request features.** Open an issue describing the use case, not just
  the solution. The "why" matters more than the "what."
- **Ask questions.** If something is unclear or undocumented, that's a
  bug in the docs. Open an issue.

## What to Expect

I maintain this in my own time. I'll read every issue, but response
times vary. If something is urgent, say so — it helps with triage.

## Security

If you've found a security vulnerability, **do not open a public issue.**
See [SECURITY.md](SECURITY.md) for reporting instructions.
