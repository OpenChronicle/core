## Execution Commands

**PowerShell Line Count Analysis:**
```powershell
Get-ChildItem core\*.py | ForEach-Object { 
    $content = Get-Content $_.FullName
    $lines = ($content | Measure-Object -Line).Lines
    $classes = ($content | Select-String "^class ").Count
    $methods = ($content | Select-String "^\s+def ").Count
    $async_methods = ($content | Select-String "^\s+async def ").Count
    
    [PSCustomObject]@{
        File = $_.Name
        Lines = $lines
        Classes = $classes
        Methods = $methods
        AsyncMethods = $async_methods
        Ratio = [math]::Round($methods / [math]::Max($lines, 1), 3)
    }
} | Format-Table -AutoSize
```

**Complexity Analysis:**
```powershell
# Install radon if not available
python -m pip install radon

# Generate complexity metrics
python -m radon cc core --total-average --show-complexity

# Maintainability index
python -m radon mi core --show
```

**Dependency Analysis:**
```powershell
# Analyze import patterns
Get-ChildItem core\*.py | ForEach-Object {
    $imports = Get-Content $_.FullName | Select-String "^(from|import)" | Where-Object { $_ -match "core\.|utilities\." }
    Write-Host "$($_.Name): $($imports.Count) internal dependencies"
    $imports | ForEach-Object { Write-Host "  $_" }
}
```
