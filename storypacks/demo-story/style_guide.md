# Demo Story Style Guide

## Overview
This is the style guide for the demo story. It contains guidelines for character dialogue, narrative style, and world-building consistency.

## Character Dialogue
- Keep dialogue natural and character-appropriate
- Use consistent speech patterns for each character
- Avoid excessive exposition in dialogue

## Narrative Style
- Use present tense for active scenes
- Maintain consistent point of view
- Keep descriptions vivid but concise

## World-Building
- Maintain consistency with established lore
- Check canon directory for established facts
- Document new world elements as they're introduced

## Tone
- Maintain appropriate tone for the story genre
- Balance serious moments with lighter elements
- Ensure emotional beats land effectively
